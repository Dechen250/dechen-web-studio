# Assets — Divina Cozinha

## Objetivo

Este documento reúne todas as referências visuais, elementos gráficos, imagens e inspirações utilizadas no projeto Divina Cozinha.

Todo desenvolvimento deve seguir estas referências.

---

# Estilo Fotográfico

As fotografias devem transmitir:

- Elegância

- Naturalidade

- Gastronomia

- Luz natural

- Ambiente acolhedor

Priorizar fotografias horizontais em alta resolução.

---

# Fotografias

## Hero

Imagem ampla mostrando um prato sofisticado sobre uma mesa elegante.

Preferencialmente:

- Massa

- Risoto

- Filé

- Frutos do mar

Sempre acompanhados por boa iluminação.

---

## Ambiente

Utilizar imagens de:

- Salão principal

- Mesas

- Iluminação quente

- Madeira

- Plantas

- Adega

- Cozinha aberta

---

## Cardápio

Fotografias profissionais dos pratos.

Categorias:

### Entradas

- Bruschettas

- Burrata

- Carpaccio

### Pratos Principais

- Massas artesanais

- Risotos

- Carnes

- Peixes

### Sobremesas

- Cheesecake

- Tiramisù

- Petit gâteau

### Bebidas

- Vinhos

- Drinks

- Café

- Espumantes

---

# Ícones

Utilizar ícones minimalistas.

Traço fino.

Outline.

Nunca utilizar ícones exagerados.

---

# Ilustrações

Evitar ilustrações.

Priorizar fotografias reais.

---

# Texturas

Utilizar discretamente:

- Papel

- Linho

- Madeira

- Cerâmica

- Mármore claro

Nunca utilizar texturas pesadas.

---

# Componentes

Os componentes devem possuir:

- Cantos arredondados

- Bordas discretas

- Sombras suaves

- Muito espaço em branco

---

# Botões

Características:

- Tons pastéis

- Hover suave

- Transições elegantes

- Bordas arredondadas

CTA principal:

Reservar Mesa

---

# Animações

Todas as animações devem transmitir elegância.

Velocidade:

Lenta.

Natural.

Suave.

Nunca utilizar animações agressivas.

---

## Cardápio

O cardápio deve possuir animação automática.

Troca de categoria:

- Entradas

- Pratos Principais

- Sobremesas

- Bebidas

Tempo:

10 segundos.

A transição deve lembrar páginas sendo viradas.

Referências de experiência:

# Responsividade

Todo componente deve funcionar perfeitamente em:

- Desktop

- Notebook

- Tablet

- Smartphone

Nenhuma animação deve prejudicar a experiência mobile.

# Objetivo Final

O visitante deve sentir que está navegando pelo site de um restaurante real, sofisticado e contemporâneo.

O projeto deve parecer uma marca consolidada e pronta para receber reservas.
