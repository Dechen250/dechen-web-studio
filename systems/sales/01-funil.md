# Funil Comercial

## Objetivo

Definir a jornada comercial utilizada pela Dechen Web Studio para transformar visitantes em clientes de forma organizada, consultiva e escalável.

Nosso objetivo não é vender rapidamente.

Nosso objetivo é compreender o negócio do cliente e construir a melhor solução possível.

# Etapa 1 — Visitante

Quem é

Pessoa que conheceu a Dechen Web Studio através de:

- Google

- Instagram

- Indicação

- Networking

- Portfólio

- Redes Sociais

Objetivo

Despertar interesse e transmitir credibilidade.

---

# Etapa 2 — Lead

Quem é

Pessoa que demonstrou interesse em nossos serviços.

Canais

- Formulário do site

- WhatsApp

- Instagram

- E-mail

- Indicação

Objetivo

Registrar o contato e iniciar o relacionamento.

Resultado esperado

Lead cadastrado no sistema.

---

# Etapa 3 — Descoberta

Objetivo

Compreender profundamente o cliente.

Perguntas principais

- Quem é a empresa?

- Quais serviços oferece?

- Quem é o público-alvo?

- Quais problemas deseja resolver?

- Existe um site atualmente?

- Quais são os objetivos do projeto?

- Existe um prazo?

- Existe um orçamento previsto?

Resultado esperado

Todas as informações necessárias para análise foram coletadas.

---

# Etapa 4 — Diagnóstico

Objetivo

Analisar as informações obtidas na Descoberta e definir a melhor estratégia.

Nesta etapa a Dechen Web Studio identifica:

- O problema principal.

- As oportunidades.

- A melhor solução.

- As prioridades do projeto.

- O escopo recomendado.

Resultado esperado

Plano estratégico definido.

---

# Etapa 5 — Proposta

Objetivo

Apresentar uma solução personalizada.

A proposta deve conter:

- Objetivos

- Escopo

- Cronograma

- Investimento

- Forma de pagamento

- Próximos passos

Resultado esperado

Cliente aprova a proposta.

---

# Etapa 6 — Cliente

Após a aprovação.

Início oficial do projeto.

Ações

- Contrato

- Pagamento inicial

- Organização do projeto

- Kickoff

Resultado esperado

Projeto iniciado.

---

# Etapa 7 — Build

Objetivo

Executar o projeto seguindo a metodologia da Dechen Web Studio.

Etapas

- Planejamento

- Design

- Desenvolvimento

- Testes

- Aprovação

Resultado esperado

Projeto pronto para publicação.

---

# Etapa 8 — Entrega

Objetivo

Publicar o projeto e garantir uma transição tranquila.

Checklist

- Deploy

- Domínio

- SEO básico

- Performance

- Testes finais

- Treinamento

Resultado esperado

Projeto entregue com sucesso.

---

# Etapa 9 — Pós-venda

Objetivo

Manter relacionamento e acompanhar resultados.

Ações

- Feedback

- Pequenos ajustes

- Suporte

- Manutenção

- Identificar novas oportunidades

Resultado esperado

Cliente satisfeito.

---

# Etapa 10 — Cliente Recorrente

Objetivo

Transformar clientes em parceiros de longo prazo.

Possibilidades

- Novos sites

- Landing Pages

- SEO

- Performance

- Manutenção

- Consultoria

- Novos projetos

Acompanhar:

- Diagnósticos concluídos

- Tempo médio de fechamento

---

# Filosofia

Cada etapa do funil existe para gerar confiança.

Não vendemos sites.

Ajudamos empresas a resolver problemas através de experiências digitais bem planejadas.

O fechamento do contrato é consequência de um processo comercial bem executado.
