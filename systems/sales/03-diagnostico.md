# Diagnóstico

## Objetivo

A etapa de Diagnóstico tem como objetivo analisar todas as informações coletadas durante a Descoberta e definir a melhor estratégia para o cliente.

Nesta etapa não desenvolvemos o projeto.

Também não apresentamos a proposta.

Nosso foco é compreender o problema, identificar oportunidades e construir uma solução estratégica.

---

# Objetivo Principal

Antes de vender qualquer serviço, devemos responder uma única pergunta:

"Qual é a melhor solução para este cliente?"

Nem sempre essa solução será o projeto inicialmente solicitado.

Nosso compromisso é recomendar aquilo que realmente gera valor.

# Análise do Negócio

Avaliar:

- Modelo de negócio

- Posicionamento

- Público-alvo

- Diferenciais

- Concorrência

- Presença digital

- Comunicação

- Objetivos

---

# Diagnóstico da Presença Digital

Analisar:

- Site atual

- Redes sociais

- Google

- SEO

- Performance

- Identidade visual

- Experiência do usuário

- Conversão

Pergunta principal

"O que está impedindo este negócio de crescer?"

---

# Problemas Identificados

Registrar:

- Problema

- Impacto

- Prioridade

Exemplo

Problema

O site é lento.

Impacto

Alta taxa de abandono.

Prioridade

Alta.

---

# Oportunidades

Registrar oportunidades de melhoria.

Exemplos

- Melhorar conversão

- Landing Pages

- SEO

- Performance

- Automações

- Integrações

- Identidade visual

- Experiência do usuário

---

# Definição da Estratégia

Após a análise, definir:

Objetivo principal

Estratégia recomendada

Escopo inicial

Prioridades

Cronograma estimado

---

# Solução Recomendada

A solução deve ser personalizada.

Exemplos

- Landing Page

- Site Institucional

- Portfólio

- Sistema interno

- Redesign

- Manutenção

- SEO

A recomendação deve resolver o problema do cliente, e não apenas atender ao pedido inicial.

---

# Critérios de Aprovação

Antes de seguir para a proposta, confirmar:

- Objetivos compreendidos

- Problemas identificados

- Escopo definido

- Prioridades estabelecidas

- Estratégia validada internamente

---

# Resultado Esperado

Ao final do Diagnóstico, a Dechen Web Studio deve possuir clareza suficiente para elaborar uma proposta personalizada, alinhada aos objetivos do cliente e baseada em uma estratégia sólida.

A proposta deve ser consequência do diagnóstico, nunca o contrário.

---

# Filosofia

Na Dechen Web Studio, acreditamos que tecnologia sem estratégia gera desperdício.

Por isso, antes de construir qualquer solução, buscamos compreender profundamente o negócio do cliente.

Nosso objetivo não é apenas entregar um site.

Nosso objetivo é gerar resultados reais.
