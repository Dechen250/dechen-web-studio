# Manifesto

A internet está cheia de sites que apenas ocupam espaço.

Nós acreditamos que um site deve representar a qualidade de uma empresa.

Um bom design não serve apenas para impressionar.

Ele deve comunicar confiança.

Uma boa interface não serve apenas para ser bonita.

Ela deve tornar cada interação intuitiva.

Tecnologia sem estratégia gera desperdício.

Design sem propósito gera decoração.

Por isso, cada projeto desenvolvido pela Dechen Web Studio combina estratégia, design e tecnologia para transformar visitantes em oportunidades.

# Princípios de Design

## Menos, porém melhor.

Cada elemento da interface precisa possuir uma função.

Se algo não melhora a experiência do usuário, ele não deve existir.

---

## Clareza acima de tudo.

O usuário nunca deve pensar.

A interface deve parecer natural.

---

## Profundidade elegante.

Utilizamos efeitos visuais para criar percepção de qualidade.

Nunca para chamar atenção sem necessidade.

---

## Movimento natural.

Toda animação deve parecer consequência da interação do usuário.

Jamais parecer artificial.

---

## Performance é Design.

Um site lento nunca será premium.

Performance faz parte da experiência.

# Sistema Liquid Glass

O conceito visual oficial da Dechen Web Studio é baseado em **Liquid Glass**.

O objetivo não é reproduzir o estilo da Apple.

É transmitir:

- Sofisticação

- Profundidade

- Tecnologia

- Elegância

## Características

• Transparência suave

• Blur moderado

• Reflexos discretos

• Brilho controlado

• Contornos sutis

• Fundo escuro

• Profundidade visual

Jamais utilizar efeitos exagerados.

O vidro deve complementar a interface.

Nunca competir com o conteúdo.

---

# Motion Design

Toda animação deve transmitir leveza.

Preferimos:

• Fade

• Scale

• Blur

• Opacity

• Translate

Jamais utilizar:

• Bounce exagerado

• Rotações desnecessárias

• Movimentos agressivos

Curva padrão:

ease-out
